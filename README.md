# GainPlan Android

Персональное Android-приложение для 12-недельного набора массы дома.

## Что уже есть

- стартовый экран с текущей неделей, фазой прогрессии и сегодняшней нагрузкой;
- минималистичный интерфейс с нижней навигацией: Главная, Упражнения, Прогресс;
- конструктор силовой тренировки: 1 упражнение из 3 вариантов на каждую группу мышц;
- недельное расписание: силовые A/B/C, волейбол, тайский бокс, восстановление;
- тренировочный таймер для подхода и отдыха;
- техника каждого упражнения: старт, движение, ошибки, прогрессия на массу;
- варианты замены упражнения на похожую группу мышц;
- расчет ориентиров калорий, белка, протеина и креатина;
- локальный прогресс: вес, талия, максимум отжиманий и подтягиваний;
- локальные напоминания в дни силовых тренировок с выбором времени;
- без аккаунта, без сервера, без входа.

## Как открыть и собрать

1. Открой папку `GainPlan` в Android Studio.
2. Дождись Gradle Sync.
3. Подключи Android-смартфон с включенной USB-отладкой.
4. Нажми Run.

Проект сделан на обычном Android SDK + Java, без внешних библиотек.

## Важное для HyperOS 3

На Xiaomi/HyperOS уведомления могут дополнительно ограничиваться системой. Если напоминания не приходят:

- разреши уведомления для GainPlan;
- разреши точные будильники/alarms, если система попросит;
- проверь автозапуск и работу в фоне;
- отключи агрессивную экономию батареи для приложения.

## Дополнительные функции, которые пока не добавлены

Я не добавлял их без подтверждения, потому что ты просил сначала спросить:

- графики веса, талии и повторений;
- журнал подходов по каждому упражнению;
- авто-прогрессия: приложение само предлагает добавить повторения/вес;
- отметка выполненной тренировки и streak;

- кастомные упражнения и редактирование плана;
- тихий режим: не напоминать, если тренировка уже отмечена выполненной;
- RPE/самочувствие после тренировки.
